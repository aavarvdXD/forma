## Forma

I've always liked to design random things using the costume editor in scratch, later, I tried Inkscape, it was surely powerful, but not as intuitive and easy to use as scratch...  

That's why, me, aavarvd, I am doing this project

**I present to you.. Forma!**

A powerful SVG editor with intuitive buttons so anyone can create professional designs without diving too much in learning to use the application!  
Written in C++ Qt, performance is not a really big deal.  

# Features
 - Shape placing
 - Node reshaping
 - Freehand drawing
 - Lines
 - Text
 - Selection tool
 - SVG export

I will continue updating this!
